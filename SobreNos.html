HTML5 / CSS3

HTML é um acrônimo para Hyper Text Markup Language. E, HTML5 é uma linguagem de marcação para estruturar e exibir conteúdo para a World Wide Web (WWW). É uma versão aprimorada do padrão HTML original que foi criado em 1990 com o objetivo de definir a plataforma Open Web.

Cascading Style Sheets (CSS) é uma linguagem de folhas de estilos usada para definir o aspecto geral e a formatação (aparência) de um documento que está escrito em uma linguagem de marcação. O CSS é interpretado por todos os navegadores da web, independentemente do sistema operacional ou do dispositivo. Ele é usado para afinar a aparência visual de uma página da web. E, CSS3 é o último padrão do CSS.

Fonte: https://blog.4linux.com.br/melhor-curso-de-html5-e-css3-por-que-aprender-html5-e-css3/

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.

Fonte: https://pt.wikipedia.org/wiki/Bootstrap_(framework_front-end)
